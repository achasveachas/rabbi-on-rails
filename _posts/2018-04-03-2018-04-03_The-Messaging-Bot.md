<h3 name="eafa" id="eafa" class="graf graf--h3 graf--leading graf--title">The Messaging Bot</h3>
<h4 name="722c" id="722c" class="graf graf--h4 graf-after--h3 graf--subtitle">Using Twilio To Add SMS Notifications To My Twitter Bot</h4>
<figure name="e9ad" id="e9ad" class="graf graf--figure graf-after--h4"><img class="graf-image" data-image-id="1*p-7Kt3Jt-YwLH7dPTjtTbA.png" src="/assets/images/posts/2018-04-03-2018-04-03_The-Messaging-Bot-0.png"></figure><p name="0393" id="0393" class="graf graf--p graf-after--figure">As you may recall, I <a href="https://blog.yechiel.me/creating-my-first-twitter-bot-b5e0da5c8cbb" data-href="https://blog.yechiel.me/creating-my-first-twitter-bot-b5e0da5c8cbb" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">recently</a> built a Twitter Bot that retweets whenever Alternate Side Parking rules are suspended in New York City (you can follow it <a href="https://twitter.com/AlterSideBot" data-href="https://twitter.com/AlterSideBot" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">here</a>).</p>
<p name="1c2b" id="1c2b" class="graf graf--p graf-after--p">The bot was a great hit; gaining over 150 followers as of the writing of this post, but it had its limitations.</p>
<p name="2bd9" id="2bd9" class="graf graf--p graf-after--p">For one, some people aren’t glued to their Twitter feeds; one retweet can easily get lost in the noise of an active feed. Then there are people who don’t have Twitter (the number one complaint when I told my family about the bot).</p>
<h4 name="6417" id="6417" class="graf graf--h4 graf-after--p">Enter Twilio</h4>
<p name="9d06" id="9d06" class="graf graf--p graf-after--h4"><a href="https://www.twilio.com/" data-href="https://www.twilio.com/" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">Twilio</a> is an API that can be used to, among other things, connect your app with a phone number that can make/receive phone calls and text messages.</p>
<p name="6b30" id="6b30" class="graf graf--p graf-after--p">I heard lots of great things about how easy it was to use, and how cool it was to make stuff with it, so I figured this would be a good project to try it out.</p>
<p name="824e" id="824e" class="graf graf--p graf-after--p">The hype was justified. The API is extremely user-friendly, the documentation is very clear, they even have a game to help you get started (try it: <a href="https://www.twilio.com/" data-href="https://www.twilio.com/" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">TwilioQuest</a>).</p>
<p name="9fdb" id="9fdb" class="graf graf--p graf-after--p">I wrote a blog post about the technical part of adding the SMS and voice functionality to my app (you can read it on Twilio’s blog <a href="https://www.twilio.com/blog/2018/04/sms-notifications-ruby-on-rails.html" data-href="https://www.twilio.com/blog/2018/04/sms-notifications-ruby-on-rails.html" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">here</a>). This post will focus on the user aspects of the app.</p>
<figure name="4836" id="4836" class="graf graf--figure graf--iframe graf-after--p"><iframe src="https://giphy.com/embed/4BiuGGfBIDWXC/twitter/iframe" width="435" height="246" frameborder="0" scrolling="no"></iframe></figure><h4 name="d889" id="d889" class="graf graf--h4 graf-after--figure">How Does It Work?</h4>
<p name="0509" id="0509" class="graf graf--p graf-after--h4">The app is very simple.</p>
<p name="e98a" id="e98a" class="graf graf--p graf-after--p">Save the following number to your phone-book: 347–404–5618</p>
<p name="ed51" id="ed51" class="graf graf--p graf-after--p">Now you can interact with my app using the following commands:</p>
<p name="6474" id="6474" class="graf graf--p graf-after--p"><strong class="markup--strong markup--p-strong">SUBSCRIBE</strong>: subscribes you to receive SMS notifications whenever Alternate Side Parking rules are suspended.</p>
<p name="34f6" id="34f6" class="graf graf--p graf-after--p"><strong class="markup--strong markup--p-strong">UNSUBSCRIBE</strong>: if you want to stop getting notifications (don’t worry, I won’t get offended).</p>
<p name="3a14" id="3a14" class="graf graf--p graf-after--p"><strong class="markup--strong markup--p-strong">STATUS</strong>: if you just want to find out the latest status of the rules (whether they are suspended or not) without subscribing</p>
<p name="ed45" id="ed45" class="graf graf--p graf-after--p">Additionally, you can also call the same number (347–404–5618) at any time to hear the latest status.</p>
<h4 name="4270" id="4270" class="graf graf--h4 graf-after--p">Spare Some Change?</h4>
<p name="dcac" id="dcac" class="graf graf--p graf-after--h4">If I thought my Twitter Bot was a success, my SMS app caught me completely unprepared. Within a day I had over 60 followers, and within a week I was at 100 (140 as of the time I wrote this post).</p>
<figure name="7263" id="7263" class="graf graf--figure graf--iframe graf-after--p"><blockquote class="twitter-tweet"><a href="https://twitter.com/yechielk/status/973563306614185984"></a></blockquote>
