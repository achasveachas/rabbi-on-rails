<h3 name="8d92" id="8d92" class="graf graf--h3 graf--leading graf--title">Explain XKCD To Me</h3>
<h4 name="d0e2" id="d0e2" class="graf graf--h4 graf-after--h3 graf--subtitle">Making My First Browser Plugin</h4>
<p name="8df6" id="8df6" class="graf graf--p graf-after--h4">I have a feeling most of my followers know about <a href="https://xkcd.com/" data-href="https://xkcd.com/" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">XKCD</a> already, the webcomic that has become a staple of nerd culture, and if you haven’t then congratulations! You are one of today’s <a href="https://www.xkcd.com/1053/" data-href="https://www.xkcd.com/1053/" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">lucky 10,000!</a></p>
<figure name="5049" id="5049" class="graf graf--figure graf-after--p"><img class="graf-image" data-image-id="0*qAe_OiCYW32UfcfO.png" data-width="462" data-height="316" src="/assets/images/posts/2018-10-16-2018-10-16_Explain-XKCD-To-Me-0.png"></figure><p name="1f87" id="1f87" class="graf graf--p graf-after--figure">I have been following XKCD religiously for years now, and have enjoyed pretty much every single one of the comics. However, occasionally, there will be one that I don’t get; perhaps an obscure math reference or a programming reference back before that became my thing. On days like that, I was thankful for <a href="https://www.explainxkcd.com/wiki/index.php/Main_Page" data-href="https://www.explainxkcd.com/wiki/index.php/Main_Page" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">Explain XKCD</a>, a wiki-like website where nerds get together to explain all of the webcomics.</p>
<figure name="4e18" id="4e18" class="graf graf--figure graf-after--p"><img class="graf-image" data-image-id="0*pLQaAaLAOh6GKsKb.png" data-width="257" data-height="257" data-is-featured="true" src="/assets/images/posts/2018-10-16-2018-10-16_Explain-XKCD-To-Me-1.png"></figure><p name="ceb5" id="ceb5" class="graf graf--p graf-after--figure">A few days ago I was looking for a small project to help me get into the world of browser plugins, and the idea occurred to me to add a link to each XKCD comic that would take a user directly to the corresponding page on Explain XKCD.</p>
<p name="5b50" id="5b50" class="graf graf--p graf-after--p">When I started, I didn’t know what was involved in making a browser plugin, but I figured I’d give it a night to get started and maybe finish it off over the weekend.</p>
<p name="f865" id="f865" class="graf graf--p graf-after--p">I ended up being surprised at how easy it was, within 2 hours of sitting down at the computer I had read the documentation, written a working plugin, packaged it, and gotten it approved on <a href="https://addons.mozilla.org/en-US/firefox/addon/xkcd-explainer/?src=search" data-href="https://addons.mozilla.org/en-US/firefox/addon/xkcd-explainer/?src=search" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">Mozilla’s add-on store!</a></p>
<figure name="3145" id="3145" class="graf graf--figure graf-after--p"><img class="graf-image" data-image-id="1*OTYUy9D2mGuf-kYjVQ_ncA.png" data-width="621" data-height="523" src="/assets/images/posts/2018-10-16-2018-10-16_Explain-XKCD-To-Me-2.png"><figcaption class="imageCaption">Notice the new button?</figcaption></figure><p name="5813" id="5813" class="graf graf--p graf-after--figure">Mozilla has <a href="https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions" data-href="https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">excellent documentation</a> that describes the entire process from beginning to end, including sample plugins you can code along.</p>
<p name="9019" id="9019" class="graf graf--p graf-after--p">The process itself is relatively straightforward. At it’s minimum all a browser plugin contains is a <code class="markup--code markup--p-code">manifest.json</code> file that contains information about the plugin (name, version, the URLs it should work on), and a JavaScript file with the actual code.</p>
<p name="be3a" id="be3a" class="graf graf--p graf-after--p">Of course, the JavaScript can get as complex as you want it to, but in my case, the whole plugin contained less than 20 lines of code.</p>
<p name="688b" id="688b" class="graf graf--p graf-after--p">If you like XKCD as much as I do I’d appreciate it if you downloaded the extension (you can find it <a href="https://addons.mozilla.org/en-US/firefox/addon/xkcd-explainer/?src=search" data-href="https://addons.mozilla.org/en-US/firefox/addon/xkcd-explainer/?src=search" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">here</a> for Firefox and <a href="https://chrome.google.com/webstore/detail/xkcd-explainer/iicndfighifkfkpijajnnhhbdmnohcml" data-href="https://chrome.google.com/webstore/detail/xkcd-explainer/iicndfighifkfkpijajnnhhbdmnohcml" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">here</a> for Chrome) and let me know how it works for you!</p>
<p name="fceb" id="fceb" class="graf graf--p graf-after--p graf--trailing">Feedback is always welcome at the <a href="https://github.com/achasveachas/xkcd-explainer" data-href="https://github.com/achasveachas/xkcd-explainer" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">GitHub project page</a>.</p>
