Yes, that’s true. I’ve mentioned it in the post (in the section titled **What Is Cloudflare?** )

That is why this solution isn’t perfect. It still protects users in certain situations. For example, a user logging in over an open wi-fi network will be protected from people snooping on the network.

Nevertheless if your website collects sensitive information (personal info, credit card payments etc.) you should not rely on something like this, instead you should set yourself up with a proper SSL certificate.
