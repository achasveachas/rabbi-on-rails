<h3 name="0bef" id="0bef" class="graf graf--h3 graf--leading graf--title">program[0] 2.0</h3>
<p name="f2d7" id="f2d7" class="graf graf--p graf-after--h3">A few weeks ago I <a href="https://blog.yechiel.me/program-0-1790417e5313#.cz2gsa8xg" data-href="https://blog.yechiel.me/program-0-1790417e5313#.cz2gsa8xg" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">wrote about</a> the first program I ever wrote. In that blogpost I spoke about the nostalgia of looking back at that horrendous, yet in a certain way beautiful, piece of code.</p>
<p name="af36" id="af36" class="graf graf--p graf-after--p">I did mention that someday I would improve it, and that someday actually came.</p>
<p name="d2c0" id="d2c0" class="graf graf--p graf-after--p">I’m sure as I learn more and grow as a programmer I will find ways to improve it even more, but for know I rewrote it and packaged it as a Ruby Gem that can be accessed on <a href="http://https://rubygems.org/gems/igros" data-href="http://https://rubygems.org/gems/igros" class="markup--anchor markup--p-anchor" rel="noopener" target="_blank">rubygems.org</a>.</p>
<p name="9495" id="9495" class="graf graf--p graf-after--p">You can also download it by running “gem install igros” in your terminal and then fire it up by simply typing “igros” (Igros Kodesh, being the name of the book I found the math trick in).</p>
<p name="b3b4" id="b3b4" class="graf graf--p graf-after--p">Enjoy!</p>
