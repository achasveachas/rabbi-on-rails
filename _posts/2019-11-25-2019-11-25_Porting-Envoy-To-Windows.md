<h3 name="9af6" id="9af6" class="graf graf--h3 graf--leading graf--title">Porting Envoy To Windows</h3>
<h4 name="cf68" id="cf68" class="graf graf--h4 graf-after--h3 graf--subtitle">My Talk At KubeCon 2019</h4>
<p name="5ba2" id="5ba2" class="graf graf--p graf-after--h4">So, I did a thing.</p>
<p name="a5aa" id="a5aa" class="graf graf--p graf-after--p">For the past few months, I’ve been working on an exciting project at Pivotal.</p>
<p name="5797" id="5797" class="graf graf--p graf-after--p">Envoy Proxy is a popular Open Source edge proxy and graduated CNCF project. The only problem is, it doesn’t work on Windows (yet).<br>Pivotal has been using Envoy for our Linux Cloud Foundry containers for a while, but we had nothing similar for Windows, so we got to work contributing an implementation back to the community.<br>Others have expressed interest in the work we were doing, so at this year’s KubeCon in San Diego, my pair Bill and I presented a progress report.</p>
