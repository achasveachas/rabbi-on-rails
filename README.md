# Rabbi on Rails

This repository contains the source for https://blog.yechiel.me.

## Found a typo or mistake?

Thank you. Corrections are very welcome.

If you found a typo, broken link, factual mistake, or unclear wording, please submit a pull request.

## How to submit a correction PR

1. Open this repository on GitHub.
2. Navigate to the post file in the _posts folder.
3. Click the pencil icon (Edit this file).
4. Make your correction.
5. In the commit message, briefly describe what you fixed.
6. Choose Create a new branch for this commit and start a pull request.
7. Open the pull request with a short summary.

That is enough. You do not need to run the site locally to submit small text fixes.

## What to include in your PR

1. A clear title, for example: Fix typo in post title from 2020-03-30.
2. A short explanation of the correction.
3. A link to the affected post URL, if available.

## Preferred scope

Small, focused pull requests are easiest to review.

Good:

1. One typo fix.
2. One factual correction.
3. One broken link fix.

Please avoid mixing unrelated fixes in a single PR.

## Not comfortable opening a PR?

Open an issue instead and include:

1. The post URL.
2. The text that should be changed.
3. The suggested replacement.
